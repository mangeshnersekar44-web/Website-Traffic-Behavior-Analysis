## 📌 Project Overview

This project analyzes website traffic data to uncover **user behavior patterns** and predict **conversion outcomes** using Machine Learning.

It simulates a real-world business scenario where companies optimize their marketing and user engagement strategies based on data insights.

---

## 🎯 Objectives

* Analyze website traffic and user behavior
* Identify key factors influencing conversions
* Build a predictive ML model
* Create an interactive dashboard for decision-making

---

## 📊 Dataset Information

The dataset contains **20,000+ user sessions** with features:

* 👤 User_ID
* 🔑 Session_ID
* 🌐 Page_Visited
* ⏱️ Time_Spent_Seconds
* 🚫 Bounce
* 📢 Traffic_Source
* 📱 Device_Type
* 🌍 Country
* 💰 Conversion

## 🤖 Machine Learning Model

**Model Used:** Random Forest Classifier

### 🎯 Objective:

Predict whether a user will convert (1) or not (0)

### 📈 Evaluation Metrics:

* Accuracy Score
* Confusion Matrix
* ROC-AUC Score

### 🔥 Feature Importance:

Identifies key drivers of conversion such as:

* Time Spent
* Traffic Source
* Device Type

## 💡 Key Insights

* ✅ Organic traffic yields higher conversions
* 📱 Mobile users dominate but convert less
* ⏱️ Higher engagement → higher conversion
* 🚫 Bounce users rarely convert
